# Idee: Second Brain Methode

**Status:** #offen #idee #produktivität

## Beschreibung

Das PARA-System von Tiago Forte als Framework für mein persönliches Wissensmanagement. MiniVault ist quasi der erste Schritt!

## PARA erklärt

- **P**rojekte – Aktive Vorhaben mit Deadline
- **A**reas – Verantwortungsbereiche (Gesundheit, Finanzen, Karriere...)
- **R**esources – Themen die mich interessieren
- **A**rchiv – Abgeschlossenes

## Wie passt das zu MiniVault?

- Projekte → Projekt-Feature nutzen (✅ bereits umgesetzt)
- Areas → Als Tags abbilden (#gesundheit, #finanzen, #karriere)
- Resources → Notizen mit #ressource Tag
- Archiv → Könnte ein eigener Filter werden
