# Idee: Geschenkefinder für den Shop

**Status:** #offen #idee #webshop

## Beschreibung

Interaktives Tool auf der Webshop-Startseite. Kunde beantwortet 3-4 Fragen, bekommt passende Produkte vorgeschlagen. Siehe [[webshop-design-ideen]].

## Flow

1. "Für wen suchst du?" → Frau / Mann / Kind / Egal
2. "Welcher Anlass?" → Geburtstag / Weihnachten / Einzug / Einfach so
3. "Budget?" → Unter 30€ / 30-60€ / Über 60€
4. → Ergebnis: 3-5 passende Produkte

## Umsetzung

- Vanilla JS, kein Framework nötig
- Produktdaten aus Shopify API
- Animierte Übergänge zwischen Fragen
- Ergebnis mit "In den Warenkorb"-Button
